1. INTRODUCCIÓN GENERAL DEL BONUS
Este bonus existe para resolver uno de los problemas más comunes, más frustrantes y más limitantes para cualquier persona que vende ropa, accesorios o productos visuales:
“Tengo la prenda, pero no tengo quién me la modele.”
Ese problema parece pequeño, pero en realidad bloquea toda la producción visual de una marca.
Porque una prenda sola puede mostrar información.Pero una prenda puesta en una persona muestra deseo.
Una camiseta en una mesa muestra algodón.Una camiseta puesta en una modelo con actitud muestra estilo.
Un vestido colgado muestra forma.Un vestido sobre un cuerpo muestra caída, movimiento, proporción y aspiración.
Un bolso sobre fondo blanco muestra producto.Un bolso en una escena con una mujer caminando muestra vida, ocasión y personalidad.
Una gorra en una foto de catálogo muestra objeto.Una gorra en un modelo con energía streetwear muestra identidad.
La ropa no se vende solo por existir.La ropa se vende cuando la persona puede imaginarse dentro de una escena.
Y ahí es donde entra el Casting Infinito IA.
Este bonus te enseña a crear modelos virtuales para tu marca usando inteligencia artificial. Pero no modelos genéricos. No caras perfectas de plástico. No influencers falsos sin alma. No imágenes bonitas que se olvidan en dos segundos.
Aquí vas a aprender a construir modelos con intención.
Modelos que sirvan para mostrar prendas.Modelos que sirvan para crear campañas.Modelos que sirvan para catálogos.Modelos que sirvan para banners.Modelos que sirvan para anuncios.Modelos que sirvan para contenido de Instagram, TikTok, web y lanzamientos.Y, en un nivel más avanzado, modelos que puedan convertirse en embajadores visuales de marca.
La promesa de este bonus es simple:
Vas a dejar de depender de tener modelos reales disponibles para poder mostrar tus prendas con fuerza visual.
No porque los modelos reales no sirvan.Sí sirven.
Pero si cada vez que quieres lanzar contenido necesitas conseguir modelo, fotógrafo, locación, maquillaje, dirección, horario, clima, presupuesto y edición, tu marca avanza lento.
Con IA puedes crear un sistema visual mucho más flexible.
Puedes crear una modelo femenina para una colección romántica.Puedes crear un hombre joven para streetwear.Puedes crear una mujer elegante para prendas premium.Puedes crear una modelo curvy para ropa más real y cercana.Puedes crear un modelo masculino para polos, hoodies o chaquetas.Puedes crear manos para accesorios.Puedes crear cuello, orejas y rostro para joyería.Puedes crear escenas completas para mostrar estilo de vida.Puedes crear una embajadora de marca con historia, personalidad y presencia.
Eso es lo que vamos a desarrollar aquí.

2. QUÉ ES REALMENTE UN CASTING INFINITO IA
Un casting infinito IA no es una carpeta llena de caras generadas al azar.
Eso no sirve.
Un casting infinito IA es una biblioteca estratégica de modelos virtuales creados para cumplir funciones visuales específicas dentro de una marca.
La palabra clave es función.
Cada modelo debe tener una razón para existir.
No creamos una modelo porque “se ve linda”.La creamos porque ayuda a vender una sensación concreta.
No creamos un hombre porque “se ve cool”.Lo creamos porque comunica el estilo que la marca quiere proyectar.
No creamos una embajadora porque “se ve profesional”.La creamos porque representa el ADN emocional, visual y comercial de la marca.
Un casting IA bien hecho responde preguntas como:
¿Qué tipo de cuerpo necesita esta prenda para verse mejor?¿Qué edad visual conecta con mi cliente ideal?¿Qué actitud hace que esta ropa se vea deseable?¿Qué energía debe tener la persona que usa esta prenda?¿Qué escena ayuda a que el producto se entienda?¿Qué nivel de realismo necesita la imagen?¿Qué modelo podría convertirse en rostro recurrente de la marca?¿Qué tipo de casting me permite producir contenido durante meses?
La mayoría de personas se queda en un prompt básico:
“Modelo usando camiseta negra, fotografía realista.”
Eso produce algo genérico.
Una persona que dirige visualmente piensa diferente:
“Necesito una modelo petite, joven, de energía suave pero segura, con pelo rizado negro como ancla visual, piel real con textura, sonrisa cálida, fondo lavanda, luz editorial suave, usando una camiseta negra charcoal de algodón con arrugas naturales, para construir un rostro de marca femenino, magnético y humano.”
Eso ya no es un prompt.
Eso es dirección creativa.

3. QUÉ PROBLEMAS RESUELVE ESTE BONUS
Este bonus resuelve problemas reales de producción.
Problema 1: No tienes modelos
Muchas marcas tienen prendas buenas, pero no tienen quién las use.
Entonces publican fotos de producto plano, prendas colgadas, fotos sobre cama, maniquíes o imágenes sin vida.
Eso puede funcionar para mostrar inventario, pero no para construir deseo.
La IA permite crear cuerpos, rostros, estilos y escenas sin depender de conseguir personas reales para cada prenda.
Problema 2: No tienes presupuesto
Una sesión real puede implicar:
modelo, fotógrafo, estudio, maquillaje, styling, transporte, edición, coordinación y tiempo.
Para una marca pequeña, eso puede ser demasiado.
Con IA puedes hacer pruebas visuales, campañas piloto y contenido base sin quemar presupuesto.
Problema 3: No tienes variedad
Una sola modelo real puede limitar la percepción de la marca.
Con IA puedes mostrar la misma prenda en:
una mujer joven,una mujer madura,una mujer curvy,una modelo editorial,una chica casual,un hombre streetwear,un modelo elegante,una persona de estética urbana,un cuerpo más atlético,un cuerpo más cotidiano.
Esa variedad ayuda a que más personas se identifiquen.
Problema 4: No puedes repetir producción cada semana
Si vendes moda, necesitas contenido constantemente.
No puedes depender de hacer una gran sesión cada tres meses y luego quedarte sin imágenes.
El casting IA permite producir piezas nuevas de forma mucho más rápida.
Problema 5: Tu marca se ve pequeña
Cuando una marca solo muestra productos aislados, se puede ver improvisada.
Cuando muestra personas, escenas, campañas, actitudes y estilo editorial, empieza a verse más grande.
El casting IA ayuda a elevar percepción.
No cambia la calidad real de la prenda, pero sí cambia la forma en que se percibe.
Problema 6: Tus prendas no tienen contexto
La ropa necesita contexto.
Una chaqueta no se entiende igual en fondo blanco que en una calle nocturna con luz editorial.
Un top no se percibe igual en una modelo rígida que en una escena de café, playa, estudio o habitación.
El contexto vende la historia de uso.

4. DIFERENCIA ENTRE MODELO IA, AVATAR IA Y EMBAJADOR DE MARCA IA
Antes de producir, hay que ordenar conceptos.
No todo es lo mismo.
4.1 Modelo IA
Un modelo IA es una persona generada para mostrar una prenda o accesorio.
Su objetivo principal es visual y comercial.
Sirve para:
mostrar fit,mostrar actitud,mostrar estilo,crear catálogo,crear anuncios,crear contenido.
Ejemplo:
Una mujer usando una blusa blanca en un estudio minimalista.Un hombre usando un hoodie oversized en una calle urbana.Una modelo mostrando aretes en primer plano.Un chico usando una gorra en una escena de skate.
El modelo IA puede aparecer una vez o varias veces.
No necesariamente necesita historia profunda.
4.2 Avatar IA
Un avatar IA es un modelo con identidad más estable.
Ya no es solo “una persona para una imagen”.
Tiene rasgos definidos y se puede repetir.
Tiene anclas visuales.
Por ejemplo:
cabello negro rizado,sonrisa pequeña,piel clara oliva,cuerpo petite,fondo lavanda,camiseta charcoal,mirada directa,textura real.
Un avatar puede usarse varias veces porque tiene identidad reconocible.
4.3 Embajador de Marca IA
Un embajador de marca IA es el nivel más avanzado.
Aquí el personaje no solo tiene rostro.
Tiene:
nombre,historia,personalidad,tono,valores,forma de hablar,frases propias,función dentro de la marca,energía narrativa,conflicto,deseo,punto de vista.
Un embajador IA puede convertirse en una figura recurrente.
Puede ser la cara de una campaña.Puede representar una colección.Puede aparecer en piezas educativas.Puede presentar productos.Puede explicar ideas.Puede construir comunidad.Puede darle humanidad a la marca.
La diferencia es esta:
Modelo IA: muestra la prenda.Avatar IA: mantiene identidad visual.Embajador IA: representa la marca.

5. LA INTENCIÓN REAL DEL BONUS
Este bonus no busca enseñar a “hacer modelos bonitos”.
Eso sería demasiado superficial.
La intención real es enseñar un sistema.
El alumno debe salir con la capacidad de:
analizar qué tipo de modelo necesita su prenda,crear una identidad visual base,generar varias imágenes del mismo modelo,mantener consistencia,crear campañas con ese modelo,crear diferentes modelos para diferentes estilos,y convertir uno de esos modelos en embajador de marca.
Aquí el alumno aprende a pensar como director creativo.
No solo escribe prompts.
Decide.
Decide qué cuerpo representa mejor la prenda.Decide qué energía vende mejor la colección.Decide qué rostro puede convertirse en ancla de marca.Decide qué estilo visual debe repetirse.Decide qué detalles hacen memorable al avatar.Decide qué elementos pueden romper la consistencia.Decide qué escenas tienen sentido comercial.
La IA ejecuta.
Pero la dirección la pone la persona.

6. PRINCIPIO CENTRAL: CREDIBILIDAD ANTES QUE PERFECCIÓN
La IA tiende a exagerar.
Si le pides belleza, te da piel plástica.Si le pides editorial, te da una imagen demasiado producida.Si le pides lujo, puede darte algo frío y falso.Si le pides modelo, puede darte una persona perfecta pero sin humanidad.
Por eso el principio central es:
Credibilidad antes que perfección.
Una imagen no vende más solo porque el modelo sea perfecto.
A veces vende menos, porque parece falsa.
El realismo visual necesita imperfecciones.
Necesita:
poros,pequeñas asimetrías,textura de piel,arrugas naturales en la ropa,cabello con mechones imperfectos,manos normales,expresión humana,postura con peso,sombras reales,luz creíble,fondos con pequeñas imperfecciones.
Una marca no necesita parecer generada por IA.
Necesita parecer producida con criterio.
La mejor imagen IA no es la que grita “mira qué espectacular”.
La mejor imagen IA es la que la persona mira y piensa:
“Esto podría ser una campaña real.”

7. LA REGLA DE LAS ANCLAS VISUALES
Un avatar consistente necesita anclas.
Las anclas visuales son elementos que se repiten y hacen reconocible al personaje.
Sin anclas, la IA cambia el rostro, el cuerpo, el pelo, la energía y la identidad en cada imagen.
Con anclas, el modelo empieza a mantenerse estable.
7.1 Qué puede ser un ancla visual
Puede ser:
tipo de pelo,color de pelo,sonrisa,forma del rostro,tono de piel,complexión,estatura,tipo de mirada,accesorio simple,prenda base,fondo base,color de marca,luz recurrente,gesto,actitud,paleta visual.
7.2 Cuántas anclas usar
Lo ideal es usar entre 3 y 5 anclas principales.
Si usas solo una, la identidad puede ser débil.
Si usas demasiadas, la IA se confunde.
Ejemplo de anclas para Andrea / Angie:
Cabello negro, rizado, abundante y brillante.
Sonrisa cálida con dientes pequeños.
Complexión petite, delgada y delicada.
Piel clara / mestiza clara con textura real.
Fondo lavanda profundo como entorno base.
Esas cinco anclas bastan para que el avatar tenga identidad.
No necesitamos tatuajes complejos, joyería difícil, logos, estampados raros o elementos imposibles de mantener.
7.3 Anclas buenas vs anclas peligrosas
Buenas anclas
Cabello definido.Color de fondo.Tipo de luz.Complexión.Forma de rostro.Sonrisa.Accesorio simple.Gafas simples.Gorra simple.Paleta de ropa.Actitud visual.
Anclas peligrosas
Tatuajes complejos.Texto en ropa.Logos reales.Joyería muy detallada.Uñas con diseños complejos.Peinados imposibles.Estampados muy específicos.Maquillaje excesivo.Manos en poses difíciles.Accesorios que cambian de forma.
La regla es:
Mientras más simple sea el ancla, más fácil será repetirla.Mientras más compleja sea el ancla, más riesgo hay de inconsistencia.

8. LAS 6 CAPAS DE UN MODELO IA PROFESIONAL
Para crear modelos IA profesionales, trabajamos con 6 capas.
No escribas prompts desde el caos.
Construye por capas.
Capa 1: Identidad física
Aquí defines el cuerpo y el rostro.
Incluye:
género,edad visual,etnicidad o mezcla visual,tono de piel,complexión,altura aproximada,rostro,ojos,nariz,labios,mandíbula,pelo,manos,postura base.
Ejemplo:
“Mujer de 23 años, piel clara mestiza, complexión petite y delgada, rostro delicado, sonrisa cálida, dientes pequeños, cabello negro rizado abundante.”
Capa 2: Realismo humano
Aquí defines qué imperfecciones debe conservar.
Incluye:
poros visibles,textura real,asimetría natural,ligera rojez,pecas suaves,líneas finas,cabello con frizz sutil,piel sin suavizado de belleza,gestos no robóticos.
Ejemplo:
“Visible pores, natural skin texture, slight redness, micro asymmetry, no beauty smoothing.”
Capa 3: Energía / actitud
Aquí defines lo que transmite.
Ejemplos:
serena,confiada,tierna,rebelde,editorial,lujosa,cercana,magnética,intelectual,juvenil,urbana,minimalista,sensual sin vulgaridad,aspiracional,amigable.
La energía debe coincidir con la marca.
No todas las marcas necesitan modelos sexys.
No todas necesitan modelos serias.
No todas necesitan modelos sonrientes.
La actitud debe vender el universo de la prenda.
Capa 4: Vestuario y tela
Aquí defines la prenda.
Incluye:
tipo de prenda,color,fit,material,caída,arrugas,textura,costuras,volumen,cómo toca el cuerpo,cómo se mueve.
Ejemplo:
“dark charcoal black crewneck t-shirt, realistic cotton folds, natural fabric texture, soft wrinkles around shoulders.”
Capa 5: Entorno y escena
Aquí defines dónde vive la imagen.
Ejemplos:
estudio blanco,fondo lavanda,apartamento real,café,calle urbana,rooftop,habitación,baño,pasarela,fondo editorial,tienda,playa,escaleras,metro,hotel,set minimalista.
El entorno no se elige porque sí.
El entorno debe reforzar la promesa de la prenda.
Capa 6: Cámara, luz y composición
Aquí defines la producción.
Incluye:
cámara full-frame,lente 35mm, 50mm, 85mm,apertura,ISO,distancia,tipo de luz,dirección de luz,temperatura,sombras,encuadre,aspect ratio,nivel de grano,retoque.
Ejemplo:
“Full-frame photography, 85mm lens, f/1.8, ISO 100, soft editorial butterfly lighting, extremely sharp focus on eyes and smile.”

10. CÓMO EXPLICAR LA PLANTILLA AL ALUMNO
Esta plantilla no es para copiar y pegar sin pensar.
Es una ficha de dirección creativa.
Cada bloque controla una parte de la imagen.
10.1 Bloque task
"task": "generate"

Aquí se indica que queremos generar una imagen nueva.
10.2 Bloque subject
Este es el bloque más importante para crear identidad.
Aquí defines a la persona.
No debes escribir solo:
“female model”
Eso es demasiado débil.
Debes definir:
tipo humano,género,edad,etnicidad visual,tono de piel,detalles del rostro,cabello,expresión y vitalidad.
La palabra vitality es muy importante.
"vitality": "alive, relaxed breathing, micro facial tension"

Esto ayuda a evitar modelos muertos, rígidos o demasiado perfectos.
Queremos que parezcan vivos.
10.3 Bloque wardrobe
Aquí defines la prenda.
No basta con decir “white t-shirt”.
Se define:
fit,tela,arrugas,caída,textura,accesorios.
La ropa debe parecer ropa real.
Una prenda IA sin arrugas, peso o textura parece disfraz digital.
10.4 Bloque environment
Aquí defines el entorno.
Para el primer avatar, lo ideal es un fondo limpio.
¿Por qué?
Porque el objetivo del primer avatar no es vender una escena.
Es fijar identidad.
Por eso se usa fondo blanco, lavanda, gris, beige o estudio.
10.5 Bloque lighting
La luz define la calidad.
Una mala luz puede hacer que la imagen se vea barata.
Una buena luz hace que el modelo parezca de editorial.
En la plantilla hay:
key light,fill light,back light,sombras.
Esto permite controlar volumen, piel, cabello y profundidad.
10.6 Bloque camera
La cámara ayuda a que la IA entienda la estética fotográfica.
No es lo mismo:
26mm smartphone,35mm calle,50mm retrato natural,85mm editorial premium.
El lente cambia la percepción del rostro.
Para identidad editorial, 85mm suele verse más premium.Para lifestyle, 35mm o 50mm se siente más natural.Para UGC, 26mm smartphone se siente más cotidiano.
10.7 Bloque angles
Este bloque sirve para crear consistencia.
No basta una imagen frontal.
Debes pedir diferentes ángulos.
Eso ayuda a que el avatar pueda funcionar en campañas reales.
10.8 Bloque realism_constraints
Este bloque es obligatorio.
Debe mantenerse casi siempre.
"no_cgi": true,
"no_illustration": true,
"no_beauty_smoothing": true,
"retain_imperfections": true,
"no_text_on_image": true

Esto protege el realismo.
10.9 Bloque post_processing
Aquí defines el acabado.
Para moda, normalmente conviene:
retoque mínimo,grano sutil,tonos neutrales,textura real.
No queremos filtros excesivos.
Queremos imagen usable.

11. PROCESO PASO A PASO PARA CREAR TU PRIMER MODELO IA
Paso 1: Define para qué necesitas el modelo
Antes de generar, responde:
¿Este modelo es para catálogo?¿Para campaña editorial?¿Para anuncio?¿Para redes?¿Para web?¿Para accesorios?¿Para una colección específica?¿Para ser rostro recurrente de marca?
No es lo mismo crear una imagen para e-commerce que para una campaña aspiracional.
Paso 2: Define el tipo de cliente que debe verse reflejado
Una marca vende más cuando la persona siente:
“Esto podría ser para mí.”
Por eso debes definir:
edad del cliente,estilo de vida,nivel aspiracional,tipo de cuerpo,ciudad o ambiente,energía,estatus,personalidad.
Ejemplo:
Si tu marca vende ropa casual femenina para chicas de 20 a 28 años, probablemente no necesitas una modelo de pasarela intimidante. Quizá necesitas una modelo natural, urbana, cercana, con belleza realista y actitud de amiga cool.
Si vendes ropa masculina premium, quizá necesitas un hombre sobrio, seguro, con postura elegante, fondo limpio y luz editorial.
Si vendes accesorios, quizá necesitas rostro, manos, cuello y orejas bien definidos.
Paso 3: Elige 3 a 5 anclas visuales
Ejemplo femenino:
cabello rizado negro,piel oliva clara,sonrisa cálida,complexión petite,fondo lavanda.
Ejemplo masculino:
barba irregular,cabello corto despeinado,piel mediterránea,sudadera lisa,balcón urbano.
Ejemplo accesorios:
manos delgadas,uñas naturales,piel real,joyería minimalista,fondo crema.
Paso 4: Crea una imagen ancla
La imagen ancla debe ser simple.
No metas todavía:
poses complejas,varios productos,fondos cargados,luces raras,ropa con texto,otros personajes.
Primero crea una foto clara del modelo.
Paso 5: Genera variaciones de ángulo
Después de tener la primera imagen, genera:
frontal,close-up,perfil,tres cuartos,plano medio,vista posterior,contact sheet.
Paso 6: Revisa consistencia
Mira si se mantiene:
rostro,pelo,piel,sonrisa,proporciones,complexión,energía.
Si cambia mucho, debes reforzar el prompt.
Paso 7: Crea escenas de campaña
Solo cuando el avatar esté estable, lo llevas a:
calle,café,habitación,estudio,lookbook,editorial,e-commerce,anuncios.
Paso 8: Integra la prenda
Sube o describe tu prenda.
Define:
color,fit,material,textura,detalles,costuras,caída,qué debe mantenerse exacto.

28. CÓMO USAR LA HISTORIA DEL AVATAR EN UNA CAMPAÑA
La historia del avatar no debe contarse completa en cada imagen.
Ese es un error.
La historia funciona como una brújula interna.
No todo se publica.
Pero todo dirige.
Cuando creas una imagen de Angie, la historia te ayuda a decidir:
cómo mira,cómo sonríe,qué ropa usa,qué tan cerca está de cámara,qué fondo tiene,qué frase puede acompañarla,qué tipo de campaña protagoniza,qué energía debe transmitir,qué marcas puede representar,qué poses no le quedan bien.
Por ejemplo:
Si Angie es “La Mujer que Nunca Subestimes”, no debería aparecer como una modelo vacía con pose exageradamente sexy.
Eso traicionaría el personaje.
Debe aparecer atractiva, sí.
Pero con inteligencia.
Con control.
Con presencia.
Con una suavidad que no parece debilidad.
La historia protege la coherencia.

Ejemplo:

¿Qué le dolió?

Puede querer:
respeto,libertad,influencia,belleza propia,pertenencia,estatus,movimiento,control,ser visto,ser recordado.

Ejemplos:
Dulce pero ambiciosa.Callado pero dominante.Elegante pero salvaje.Cercana pero selectiva.Minimalista pero intensa.Juvenil pero estratégica.Fría por fuera, emocional por dentro.
29.8 Forma de hablar
Define:

29.9 Estética
Define:
colores,prendas,telas,fondos,luz,ángulos,tipo de cámara,tipo de poses.
29.10 Función comercial
¿Para qué existe dentro de la marca?
Puede existir para:
presentar colecciones,ser rostro de anuncios,modelar prendas,crear confianza,educar,inspirar,vender aspiración,hacer contenido lifestyle,humanizar la marca.

39. CÓMO CREAR VARIEDAD SIN PERDER IDENTIDAD
La consistencia no significa que todas las imágenes sean iguales.
Significa que el personaje se reconoce aunque cambie la escena.
Puedes variar:
ropa,pose,formato,fondo,luz,expresión,distancia,ángulo.
Pero debes mantener:
rostro,edad visual,pelo,complexión,tono de piel,energía base.
Regla práctica
Si vas a cambiar mucho la ropa, mantén el fondo y luz parecidos.
Si vas a cambiar el fondo, mantén ropa y pose simples.
Si vas a cambiar la expresión, mantén ángulo y luz similares.
Si vas a cambiar el ángulo, mantén pelo y fondo muy claros.
No cambies todo al mismo tiempo.

40. CÓMO CREAR UN CASTING COMPLETO PARA UNA MARCA DE ROPA
Una marca puede crear un casting base con 5 tipos de modelos.
Modelo 1: Rostro principal
Es el embajador o embajadora.
Aparece en campañas importantes.
Tiene historia, personalidad y anclas fuertes.
Modelo 2: Modelo de catálogo
Más neutral.
Sirve para mostrar prendas sin robar protagonismo.
Debe ser claro, limpio y replicable.
Modelo 3: Modelo lifestyle
Sirve para escenas reales.
Café, calle, apartamento, viajes, rutina.
Modelo 4: Modelo aspiracional
Más editorial.
Sirve para banners, campañas premium y lanzamientos.
Modelo 5: Modelo de detalle
No necesariamente muestra rostro completo.
Sirve para manos, cuello, espalda, textura, accesorios.
Con estos 5 modelos, una marca pequeña puede producir muchísimo contenido.

42. ERRORES COMUNES Y CÓMO CORREGIRLOS
Error 1: Modelo demasiado perfecto
Se ve falso.
Corrección:
Agregar:
visible pores,natural skin texture,slight redness,micro asymmetry,no beauty smoothing.
Error 2: El avatar cambia en cada imagen
Corrección:
Reforzar anclas:
same identity,same face structure,same hair texture,same skin tone,same small teeth,same body proportions.
Error 3: La ropa se ve inventada
Corrección:
Describir prenda exacta:
fabric,fit,seams,collar,sleeves,texture,color,no logo changes,no added text.
Error 4: Las manos salen deformes
Corrección:
Pedir manos naturales:
realistic hands,five fingers,natural knuckles,relaxed hand pose,no distorted fingers.
También evitar poses complicadas al inicio.
Error 5: El fondo roba protagonismo
Corrección:
Usar fondos simples.
Primero estudio.
Después lifestyle.
Error 6: La imagen parece IA
Corrección:
Pedir:
raw photographic realism,subtle grain,minimal retouching,realistic shadows,natural imperfections,no CGI,no illustration.
Error 7: La modelo no ayuda a vender la prenda
Corrección:
Volver a la pregunta:
¿Qué energía necesita esta prenda?
No todos los productos necesitan el mismo tipo de persona.

43. GUION DE CLASE PARA EXPLICAR ESTE BONUS
Apertura
En este bonus vamos a resolver un problema muy concreto: tener ropa, pero no tener quién la modele.
Porque una prenda sin cuerpo puede verse bien, pero una prenda sobre una persona genera contexto, deseo y aspiración.
La inteligencia artificial nos permite crear modelos virtuales, pero el objetivo no es generar caras bonitas al azar.
El objetivo es crear un casting visual que pueda trabajar para tu marca.
Vamos a aprender a crear modelos femeninos, masculinos, modelos para accesorios y, en un nivel más avanzado, embajadores de marca con identidad, historia y personalidad.
Parte 1: Mentalidad
Antes de abrir cualquier herramienta, entiende esto:
Un modelo IA no es decoración.
Es una decisión comercial.
Cada rostro, cuerpo, pose, escena y luz debe responder a una intención.
Si vendes ropa casual, necesitas cercanía.Si vendes ropa premium, necesitas aspiración.Si vendes streetwear, necesitas actitud.Si vendes accesorios, necesitas detalle.Si quieres construir marca, necesitas repetición y memoria.
Parte 2: Anclas
Para que un avatar se mantenga, necesitas anclas visuales.
No intentes poner demasiados detalles.
Elige de 3 a 5 elementos fuertes.
Por ejemplo:
cabello,sonrisa,tono de piel,complexión,fondo,estilo de ropa.
Eso permite que la IA repita la identidad.
Parte 3: Imagen ancla
Primero crea una imagen simple.
Fondo limpio.
Rostro claro.
Luz controlada.
Ropa básica.
Nada complejo.
Esa imagen será el origen.
Parte 4: Pack de ángulos
Después generas:
frontal,close-up,perfil,tres cuartos,ángulo alto,ángulo bajo,vista posterior,plano medio,contact sheet.
Esto te da un avatar más fuerte.
Parte 5: Historia
Si quieres convertirlo en embajador de marca, debes darle vida.
Nombre.Historia.Psicología.Tono.Frases.Estética.Función comercial.
Un avatar sin historia puede verse bonito.
Un avatar con historia puede volverse memorable.
Parte 6: Campaña
Finalmente llevas el avatar a campaña.
Lo vistes con tu prenda.
Lo pones en una escena.
Creas formatos.
Y produces contenido para redes, web, anuncios y catálogo.
